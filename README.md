🚀 ResumePro – AI Powered Resume Builder (MERN Stack)

ResumePro is a full-stack AI-powered resume builder platform that helps users create industry-level professional resumes with intelligent content suggestions.
The platform provides role-based dashboards for users, recruiters, and administrators, enabling efficient resume creation, management, and interaction.

✨ Key Features

✅ AI Resume Assistance
Generate and improve resume sections like summary, skills, and experience using AI.

✅ Interactive Resume Builder
Dynamic form-based interface for building professional resumes quickly.

✅ Role-Based Dashboards
Separate dashboards for Users, Recruiters, and Admins.

✅ Real-Time Features
Implemented Socket.IO for instant communication and notifications.

✅ Authentication & Authorization
Secure login and user access control.

✅ Responsive UI
Works across desktops, tablets, and mobile devices.
